# election-polarization-analysis
# Analyse de la Polarisation électorale

## Description 
Ce projet implémente et compare plusieurs mesures de polarisation pour des profils
électoraux dans deux types de votes :
- **Votes par approbation** : vecteur binaire indiquant les candidates approuvées
- **Votes par ordres totaux** : classement complet des candidates par ordre de préférence

## Prérequis

Python 3.8+ 
Les Bibliothèques suivantes : 
numpy - matplotlib, scipy 
--> Pour les installer : pip install numpy matplotlib scipy

## Lancer le projet
sur la console: python election.py OR py election.py
